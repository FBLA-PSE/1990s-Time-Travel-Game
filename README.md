# 1990s-Time-Travel-Game
FBLA 1990s Time Travel Game
Made by Preston Chua, Elvin Pio, and Seiyon Ravishankar

## About the Project
This game was developed in Code.org’s Game Lab using JavaScript. It was created for the FBLA Middle School competition. The goal was to design a functional, engaging game that demonstrates programming skills, creativity, and usability, while also providing information about historic events.

---

## How to Play

1. Open the project link:
   https://studio.code.org/projects/gamelab/1b9500fd-4f91-4b46-9249-f55ba0d8559b

2. Click "Run" to start the game.

3. Use the keyboard controls (such as arrow keys, if implemented) to move and interact.

Complete the objectives in each scene to progress.

If you lose on the Mars Rover level, you will have to refresh the browser to restart.
